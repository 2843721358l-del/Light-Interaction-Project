# Light Interaction Project Page

This is a static project page for the paper:

**Light Interaction: Training-Free Inference Acceleration for Interactive Video World Models**

## Local Preview

Open `index.html` directly in a browser.

## GitHub Pages

1. Create a GitHub repository, for example `Light-Interaction`.
2. Upload everything inside this `project_page` folder to the repository root.
3. In the repository settings, enable GitHub Pages from the `main` branch and root directory.
4. Replace the disabled Paper, arXiv, and Code links in `index.html` once they are available.
